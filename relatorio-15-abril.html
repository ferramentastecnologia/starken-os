4. Melhorias no Sistema (Starken OS)

Novas Funcionalidades

Minhas Tasks — Filtro por role: Executor (Dante) agora só vê tasks com status Agendar e Aprovação
Dashboard — Agendamentos por dia: Seção agora agrupa por dia da semana (SEG, TER...) e por cliente com tasks individuais
Dashboard — Progresso Semanal/Mensal: Dados reais do Supabase (publicados · agendados · total)
Colunas redimensionáveis: Arrastar bordas das colunas no Minhas Tasks (estilo Finder do Mac)
Date picker visual: Campo de data abre calendário nativo do navegador (sem digitação)
Seção "Últimos 7 dias": Nova seção no Minhas Tasks mostra tasks agendadas/publicadas recentes
Auto-mudança de status: Ao agendar/publicar, status muda automaticamente (Aprovação → Agendado/Publicado)
Auto-promoção por hora: Tasks agendadas mudam para Publicado quando o horário exato passa
PRD + Design System: Documentação completa do produto para redesign no Stitch
