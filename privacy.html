Política de Privacidade

Starken OS — Sistema de Gestão de Marketing Digital
Última atualização: 24 de março de 2026

1. Informações Gerais

O Starken OS é um sistema interno de gestão de marketing digital operado pela Starken Assessoria Ltda, CNPJ 62.839.769/0001-55 ("nós", "nosso"). Esta política descreve como coletamos, usamos e protegemos informações ao utilizar nosso sistema e integrações com plataformas de terceiros, incluindo Meta (Facebook e Instagram).

2. Dados que Coletamos

Nosso sistema coleta e processa os seguintes tipos de dados:

  • Dados de contas Meta: Informações públicas de páginas do Facebook, contas do Instagram e contas de anúncios vinculadas ao Business Manager do cliente, incluindo nomes, IDs, métricas de desempenho e insights de campanhas.
  • Conteúdo de publicações: Textos, imagens e vídeos que os usuários do sistema fazem upload para publicação nas redes sociais dos clientes.
  • Dados de campanhas publicitárias: Métricas de performance (impressões, cliques, alcance, conversões, gastos) obtidas via Meta Marketing API.
  • Dados de usuários do sistema: Nome e PIN de acesso dos operadores internos (equipe da agência).
  • Dados de clientes da agência: Informações comerciais como nome do negócio, segmento, contatos, links de redes sociais e materiais de identidade visual.

3. Como Usamos os Dados

Utilizamos os dados exclusivamente para:

  • Gerenciar e publicar conteúdo nas redes sociais dos clientes da agência.
  • Agendar publicações no Facebook e Instagram.
  • Monitorar e analisar o desempenho de campanhas publicitárias.
  • Gerar relatórios de performance para os clientes.
  • Gerenciar campanhas de anúncios (criação, edição, pausar/ativar).
  • Produzir cronogramas de conteúdo em formato PDF.

4. Integrações com Terceiros

O Starken OS integra-se com os seguintes serviços:

  • Meta Graph API (v25.0): Para publicação de conteúdo, leitura de insights e gerenciamento de anúncios no Facebook, Instagram e Threads.
  • Supabase: Para armazenamento seguro de dados e arquivos do sistema.
  • Vercel: Para hospedagem e execução do sistema.

Não compartilhamos dados com terceiros além dos serviços necessários para o funcionamento do sistema.

5. Armazenamento e Segurança

  • Os dados são armazenados em servidores seguros do Supabase (PostgreSQL com criptografia em trânsito e em repouso).
  • Tokens de acesso à Meta API são armazenados de forma segura como variáveis de ambiente no servidor.
  • O acesso ao sistema é restrito a usuários autorizados da equipe via autenticação por PIN.
  • Arquivos de mídia são armazenados no Supabase Storage com acesso controlado.

6. Retenção de Dados

Os dados são mantidos enquanto o cliente mantiver contrato ativo com a agência. Dados de clientes encerrados podem ser removidos mediante solicitação.

7. Direitos dos Usuários

Em conformidade com a Lei Geral de Proteção de Dados (LGPD — Lei 13.709/2018), os titulares de dados têm direito a:

  • Solicitar acesso aos seus dados pessoais.
  • Solicitar correção de dados incorretos.
  • Solicitar exclusão de dados pessoais.
  • Revogar consentimento a qualquer momento.

8. Uso da Meta Platform

O uso de dados obtidos através das APIs da Meta está sujeito à Política da Plataforma Meta. Nós:

  • Não vendemos dados obtidos da Meta a terceiros.
  • Não utilizamos dados da Meta para fins de vigilância.
  • Não transferimos dados da Meta para data brokers.
  • Limitamos o uso dos dados ao escopo autorizado pelas permissões concedidas.
  • Excluímos dados da Meta quando solicitado pelo usuário ou quando o acesso é revogado.
